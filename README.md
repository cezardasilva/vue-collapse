> A simple collapse component for vue.js

# Instalation
```bash
npm install vue-collapse --save-dev
```

# Usage
```Vue
<template>
    <div>
        <collapse :selected="true">
			<div slot="collapse-header">

			</div>
			<div slot="collapse-body">

			</div>
		</collapse>
    </div>
</template>

<script>

import Collapse from 'vue-collapse'
export default {
	name: "YourComponent",
	components: {
	    Collapse
	}
}
</script>
```
